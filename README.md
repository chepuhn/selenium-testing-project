Проект автоматизированного тестирования веб-приложения

Для запуска:
1. Установить зависимости:
   pip install -r requirements.txt

2. Запустить тесты:
   pytest -v

Основные сценарии:
- Авторизация
- Добавление товара в корзину
- Оформление заказа
- Сортировка
- Выход из системы
- Тесты оплаты (mock)

Дополнительно:
- Пример неуспешного теста (demo_failures)
